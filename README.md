# 🛡️ PhishShield – AI-Powered Phishing Detector

PhishShield is a premium, real-time phishing domain detection platform designed to protect cryptocurrency users from malicious domains, typosquatting, and homoglyph attacks. Leveraging state-of-the-art neural patterns and AI-driven analysis, PhishShield provides immediate verdicts on the safety of any URL.

## 🚀 Features

- **Autonomous Neural Analysis**: Real-time inspection of URLs using advanced AI models.
- **Typosquatting Detection**: Identifies domains designed to mimic popular platforms (e.g., `coinbace.com` vs `coinbase.com`).
- **Homoglyph Protection**: Detects visually similar characters used in malicious URLs.
- **Live Threat Feed**: Continuous monitoring of active phishing nodes across the web.
- **Premium UI/UX**: Engineered with a modern, glassmorphic interface and GSAP-powered animations.
- **Actionable Advice**: Provides clear instructions on how to handle detected threats.

## 🛠️ Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/) (App Router)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [GSAP](https://greensock.com/gsap/) (GreenSock Animation Platform)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Deployment**: [Render](https://render.com/) / [Vercel](https://vercel.com/)

## 🏁 Getting Started

### Prerequisites

- Node.js 18.x or later
- npm / yarn / pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/phishshield.git
   cd phishshield
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   Create a `.env.local` file in the root directory and add your API keys:
   ```env
   CLAUDE_API_KEY=your_api_key_here
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
